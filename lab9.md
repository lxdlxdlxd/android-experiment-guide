4. 对于结算结果的音频播放：（1）先把计算结果转成字符串型，再启动一个子线程来处理该字符串；（2）子线程获取传过来的计算结果的字符串后，先停止播放的“等于”对应的音频，然后利用for循环语句和String类的substring()方法把计算结果一位一位的截取出来，并播放对应的音频文件。

class Test extends Thread{
        private String result;  //计算的结果
        public Test(String s){
            this.result = s;
        }
        @Override
        public void run() {
            String s;
            sp.stop(spMap.get("equal"));
            // 用for循环把计算结果分割成对应spMap中的key
            for(int i=0; i<result.length(); i++){
                try {
                    Thread.sleep(350);  //该线程睡350毫秒
                } catch (InterruptedException e) {
                    e.printStackTrace();
                }
                s = result.substring(i, i+1);
                sp.play(spMap.get(s), 1, 1, 0, 0, 1);
            }
        }
    }
